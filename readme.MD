# Minecraft Server

Fabric server for Minecraft 1.21.1

## Starting the Server

1. Navigate to the server directory:
```bash
cd server
```

2. Accept the EULA by editing `eula.txt` and changing `eula=false` to `eula=true`

3. Run the start script:
```bash
./start.sh
```

The server will start with 2GB of memory allocated.
